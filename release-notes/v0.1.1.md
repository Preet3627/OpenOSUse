## v0.1.1 — Accessibility Tree, MCP Protocol & Liquid Glass UI

This release adds three major capabilities to OpenOSUse:

### 🌳 Accessibility Tree Integration
The agent can now read the structured Accessibility element tree of the frontmost application — buttons, text fields, menus, sliders, and their positions, titles, and states — all returned as JSON. This gives the AI model structured UI context instead of relying solely on pixel analysis.

### 🔌 MCP Protocol Server
OpenOSUse now speaks the Model Context Protocol (MCP) over TCP. Any MCP‑compatible client can connect to port 8081 and invoke tools: `screenshot`, `axTree`, `click`, `type`, `open_app`. This enables remote agent orchestration and multi‑agent setups.

### ✨ Liquid Glass UI Redesign
The dashboard has been completely redesigned with macOS Tahoe glass material aesthetics:
- Sidebar‑adaptable navigation
- Floating control panels
- Shimmer loading states for agent steps
- Animated transitions between states
- Refined SF Pro typography throughout
